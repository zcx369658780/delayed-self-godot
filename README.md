# 迟到的我 / Delayed Self

一款离线单人、2D 俯视网格回合制解谜游戏：玩家与数回合前的时间回声协作，利用“迟到”的行动开启通路并拼回一条求救信息。

本项目面向 GodotHub Festival 2026，主题为 **Delay / 滞后**。当前状态为 **pre-production / repository bootstrap**；仓库尚无 Godot 项目或可玩实现。
