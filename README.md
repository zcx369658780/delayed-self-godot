# 迟到的我 / Delayed Self

一款离线单人、2D 俯视网格回合制解谜游戏：玩家与数回合前的时间回声协作，利用“迟到”的行动开启通路并拼回一条求救信息。

本项目面向 GodotHub Festival 2026，主题为 **Delay / 滞后**。当前仓库包含一个可运行的八关 Godot 内部玩法基线：第 8 关为唯一正式终局，并带有完成确认与返回流程。持久化存档、设置、本地化、授权字体/素材、Windows 导出、RC、发布和投稿仍未完成，不能据此作出发布或提交声明。
